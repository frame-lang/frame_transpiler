### Working Examples

#### Module Declaration
```frame
module utils {
    fn helper(x) {
        return x * 2
    }
    
    var version = "1.0"
}

fn main() {
    var result = utils.helper(5)  // Works!
    print("Version: " + utils.version)  // Works!
}
```

#### Nested Modules
```frame
module lib {
    module math {
        fn square(x) {
            return x * x
        }
    }
}

fn main() {
    var result = lib.math.square(5)  // Returns 25
}
```

#### Historical Note
In current releases, the FSL abstraction has been removed. Use native imports and MixedBody (or FID manifests) instead of `from fsl import …`.

## Migration from v0.33

### Before (v0.33)
```frame
// FSL available by default
fn main() {
    var s = str(42)  // Worked without import
}

// Functions at global scope
fn helper() { }
fn utility() { }
```

### After (v0.34)
```frame
// Must import FSL
from fsl import str

// Organize in modules
module utils {
    fn helper() { }
    fn utility() { }
}

fn main() {
    var s = str(42)  // Requires import
    utils.helper()  // Qualified access
}
```
