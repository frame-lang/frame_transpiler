## 🔄 Before/After Comparison

### Previous Approach
```typescript
// Inline complexity - 15+ lines
((l, r) => {
    if (l instanceof Set && r instanceof Set) {
        return l.size === r.size && [...l].every(x => r.has(x));
    }
    if (Array.isArray(l) && Array.isArray(r)) {
        return l.length === r.length && l.every((x, i) => x === r[i]);
    }
    return l === r;
})(set1, set2)
```

### Runtime Approach
```typescript
// Clean, maintainable - 1 line
FrameRuntime.equals(set1, set2)
```
