## Key Fixes in v0.86.10

### 1. Dictionary Unpacking Corrected
- Fixed `{**base}` generating `{...base: null}`
- Parser was placing DictUnpackExprT as key, not value
- Now correctly generates `{...base}`

### 2. Lambda Expression Cleanup
- Removed double parentheses: `((x, y))` → `(x, y)`
- Single parameter lambdas: `x => ...`
- Multiple parameters: `(x, y) => ...`

### 3. String Multiplication Refinement
- Conservative detection - only for literal strings
- Fixed `x * 2` being treated as `(x).repeat(2)`
- Preserves numeric multiplication in comprehensions
