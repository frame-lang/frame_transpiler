# Frame Transpiler v0.86.39

- Fixes Bug #070 for Python V3 modules:
  - Import scanning no longer consumes `import` statements inside `actions:` bodies as module imports.
  - Outline scanning recognizes `async` handlers/actions in `machine:` and `actions:` sections.
  - Python module compilation of `runtime_protocol.frm` now emits both handlers and `_action_*` methods, as required by the debugger harness.
