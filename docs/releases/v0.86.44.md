# Frame Transpiler v0.86.44

- Fixes **Bug #075** for the TypeScript V3 expander:
  - Each transition expansion is now wrapped in its own block and uses a scoped temporary (`__frameNextCompartment`) so multiple transitions inside a single handler (e.g., across switch cases) do not trigger `TS2451` redeclaration errors.
- Adds a TypeScript V3 CLI regression fixture (`redeclare_next_compartment.frm`) that uses `@tsc-compile` to confirm that generated modules with multi-state handlers compile cleanly under `tsc`.
