# Frame Transpiler v0.86.43

- Fixes **Bug #074** for the TypeScript V3 runnable module path:
  - Emits domain variables from `domain:` as concrete class properties (with initializers when present).
  - Emits actions and operations as class methods so state handlers can call `this.handle(...)`, `this.enqueueCommand(...)`, etc. without TS2339.
  - Propagates interface/handler parameters into method signatures (e.g., `runtimeMessage(payload)` → `runtimeMessage(__e, compartment, payload)`), eliminating unbound identifiers like `payload` in generated code.
- Adds TypeScript V3 CLI regression fixtures that use `@tsc-compile` to ensure the generated modules compile under `tsc` in the test harness.
