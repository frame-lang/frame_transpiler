# Frame v0.38 - Dictionary Features Implementation

## Overview
Frame v0.38 introduces comprehensive dictionary support including dictionary comprehensions, merging operations, and full compatibility with Python's collections module special dictionary types.

## Implemented Features

### 1. Dictionary Comprehensions
Frame now supports Python-style dictionary comprehensions with full conditional filtering:

```frame
// Basic dictionary comprehension
var squares = {x: x * x for x in range(5)}
// Result: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

// With conditional filtering
var even_squares = {x: x * x for x in range(10) if x % 2 == 0}
// Result: {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

// Using unpacking in comprehensions
var combined = {k: v for k, v in [("a", 1), ("b", 2)]}
// Result: {"a": 1, "b": 2}
```

### 2. Dictionary Merging Methods

#### dict.update() - In-place Modification
```frame
var d1 = {"a": 1, "b": 2}
var d2 = {"c": 3, "d": 4}
d1.update(d2)  // d1 is now {"a": 1, "b": 2, "c": 3, "d": 4}
```

#### Dictionary Unpacking with ** Operator
```frame
var dict1 = {"a": 1, "b": 2}
var dict2 = {"c": 3, "d": 4}
var merged = {**dict1, **dict2}  // Creates new dict
```

#### Union Operator | (Python 3.9+)
```frame
var d1 = {"a": 1, "b": 2}
var d2 = {"c": 3, "d": 4}
var result = d1 | d2  // Creates new dict
```

#### dict.fromkeys() - Create with Default Values
```frame
var keys = ["x", "y", "z"]
var d = dict.fromkeys(keys, 0)
// Result: {"x": 0, "y": 0, "z": 0}
```

### 3. Special Dictionary Types (collections module)

All special dictionary types from Python's collections module are fully supported through Frame's import system:

#### defaultdict - Auto-creates Missing Values
```frame
from collections import defaultdict

var dd_int = defaultdict(int)
dd_int["count"] = dd_int["count"] + 1  // No KeyError

var dd_list = defaultdict(list)
dd_list["items"].append("apple")
```

#### OrderedDict - Maintains Insertion Order
```frame
from collections import OrderedDict

var od = OrderedDict()
od["first"] = 1
od["second"] = 2
od.move_to_end("first")  // Reorder items
```

#### Counter - Counts Hashable Objects
```frame
from collections import Counter

var words = ["apple", "banana", "apple"]
var count = Counter(words)
// Result: Counter({'apple': 2, 'banana': 1})

var most_common = count.most_common(2)
```

#### ChainMap - Chains Multiple Mappings
```frame
from collections import ChainMap

var defaults = {"color": "red", "size": "medium"}
var user_prefs = {"color": "blue"}
var settings = ChainMap(user_prefs, defaults)
// settings["color"] returns "blue" (from user_prefs)
// settings["size"] returns "medium" (from defaults)
```

## Implementation Details

### AST Changes
- Added `DictComprehensionNode` to represent dictionary comprehensions
- Added `BitwiseOr` to `OperatorType` enum for union operator support

### Parser Enhancements
- New `dict_comprehension()` method for parsing comprehension syntax
- Added `bitwise_or()` parsing level for union operator precedence
- Enhanced pattern matching for dictionary-specific expressions

### Visitor Updates
- Python visitor generates proper dictionary comprehension syntax
- Handles union operator (|) for dictionary merging
- Preserves all native Python dictionary methods through passthrough

## Known Limitations

1. **Lambda syntax not supported**: Frame doesn't support lambda expressions, use regular functions instead
2. **Compound assignment operators**: `|=` not supported, use `d1 = d1 | d2` as workaround
3. **Nested functions**: Functions inside functions not supported

## Usage Examples

### Complete Example: User Session Tracking
```frame
from collections import defaultdict, Counter

fn track_sessions() {
    // Track user visits with defaultdict
    var visits = defaultdict(int)
    visits["alice"] = visits["alice"] + 1
    visits["bob"] = visits["bob"] + 1
    visits["alice"] = visits["alice"] + 1
    
    // Count page views with Counter
    var pages = ["home", "about", "home", "contact", "home"]
    var page_counts = Counter(pages)
    var top_pages = page_counts.most_common(2)
    
    // Merge user preferences
    var defaults = {"theme": "light", "lang": "en"}
    var user_prefs = {"theme": "dark"}
    var settings = {**defaults, **user_prefs}
    
    // Create summary using dict comprehension
    var active_users = {user: count for user, count in visits.items() if count > 1}
    
    print("Active users: " + str(active_users))
    print("Top pages: " + str(top_pages))
    print("User settings: " + str(settings))
}
```

## 7. Loading from External Sources

Frame v0.38 supports loading dictionaries from various external sources through Python's standard library:

### JSON Operations
```frame
import json

// Parse JSON string to dictionary
var json_str = "{\"name\": \"Alice\", \"age\": 30}"
var d = json.loads(json_str)

// Convert dictionary to JSON string
var data = {"user": "Bob", "score": 95}
var json_output = json.dumps(data)

// Pretty printing with indentation
var pretty = json.dumps(data, indent=2)

// File operations (without 'with' statement support yet)
var f = open("data.json", "w")
json.dump(data, f, indent=2)
f.close()

var f2 = open("data.json", "r")
var loaded = json.load(f2)
f2.close()
```

### Configuration Files
```frame
import configparser

// Read INI configuration files
var config = configparser.ConfigParser()
config.read("config.ini")

// Convert sections to dictionaries
var db_config = dict(config["database"])
print("DB Host: " + db_config["host"])

// Access all sections
var sections = config.sections()
```

### Environment Variables
```frame
import os

// Get single environment variable
var home = os.environ.get("HOME", "default")

// Convert all environment variables to dictionary
var env_dict = dict(os.environ)
print("Total env vars: " + str(len(env_dict)))

// Check specific variables
var user = os.environ.get("USER", "unknown")
```

## 8. Dynamic Dictionary Creation

Frame v0.38 supports dynamic dictionary creation patterns for building complex data structures at runtime.

### Using setdefault

The `setdefault` method provides a concise way to initialize dictionary values:

```frame
var d = {}

// Traditional approach (verbose)
var fruits = d.get("fruits", [])
fruits.append("apple")
d["fruits"] = fruits

// Using setdefault (concise)
d.setdefault("fruits", []).append("apple")
d.setdefault("fruits", []).append("banana")

// Building nested structures
var data = {}
var items = [["fruits", "apple"], ["fruits", "banana"], ["vegetables", "carrot"]]
var i = 0
while i < len(items) {
    var category = items[i][0]
    var item = items[i][1]
    data.setdefault(category, []).append(item)
    i = i + 1
}
```

### Safe Alternatives to eval()

Frame supports safe methods for converting strings to dictionaries:

```frame
import json
import ast

// JSON parsing (most portable)
var json_str = "{\"name\": \"Alice\", \"age\": 30}"
var d = json.loads(json_str)

// Python literal evaluation (safe for literals only)
var py_str = "{'name': 'Bob', 'age': 25}"
var d2 = ast.literal_eval(py_str)

// Manual parsing for custom formats
var custom = "key1=value1,key2=value2"
var d3 = {}
var pairs = custom.split(",")
var i = 0
while i < len(pairs) {
    var kv = pairs[i].split("=")
    if len(kv) == 2 {
        d3[kv[0]] = kv[1]
    }
    i = i + 1
}
```

**WARNING**: Never use `eval()` with untrusted input as it can execute arbitrary code!

## Summary

Frame v0.38 provides comprehensive dictionary support that seamlessly integrates with Python's dictionary features. The implementation maintains Frame's design philosophy while enabling full access to Python's powerful dictionary capabilities, including:

- Dictionary comprehensions with conditional filtering
- Modern merging operations (update, unpacking, union operator)
- Specialized dictionary types from the collections module
- Loading from external sources (JSON, config files, environment variables)
- Dynamic dictionary creation with setdefault
- Safe string-to-dictionary conversion methods

All these features work natively through Frame's import system and expression handling.
