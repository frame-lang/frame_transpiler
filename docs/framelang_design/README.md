## Frame Interface Definitions (`.fid`)

Generated native module contracts now use the `.fid` extension (Frame Interface Definition). These files play the same role as C headers: they describe the functions and types a target runtime exposes so the Frame compiler can validate usage. Autogenerated `.fid` files live in the build cache and should not be hand-edited.
